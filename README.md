# MHB Ouders Weert — Landingspagina

Statisch prototype van de publieke landingspagina voor ouders van hoogbegaafde kinderen in de regio Weert/Nederweert.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- lucide-react, qrcode.react

## Ontwikkelen

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Content

Alle copy staat in `lib/content.ts`. Vervang `WHATSAPP_INVITE` met de echte invite-URL wanneer beschikbaar.

## Deploy

Push naar `main` op GitHub — Vercel deployt automatisch.
